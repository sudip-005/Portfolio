# 🌿 Sudip Manna — 3D Developer Portfolio

🔗 **Live Demo:** [sudip-3d-portfolio.web.app](https://sudip-3d-portfolio.web.app/)

Welcome to my interactive **3D Developer Portfolio**, a modern web experience designed to showcase my projects, development stack, and academic journey. Built at the intersection of technology and design, this sanctuary represents a blend of cinematic 3D visuals, smooth micro-animations, and organic nature-inspired vibes.

---

## ✨ Features

- **🎬 Immersive 3D Scene Layer**: Powered by React Three Fiber and Three.js, rendering interactive 3D elements dynamically.
- **🎵 Ambient Soundscapes**: Implements birds chirping for daytime, crickets for night, and relaxing background meditation music, featuring smooth volume crossfading and custom toggle controls.
- **🌓 Dynamic Day & Night Modes**: A cinematic 1.5-second transition between light and dark themes using cubic easing (`easeInOutCubic`) and state-based adjustments.
- **⏱️ System Clock Pill**: A floating digital system clock that updates dynamically in real-time.
- **📱 Premium Responsive UI**: Fully optimized layout utilizing Next.js 16, React 19, Tailwind CSS v4, Framer Motion, and GSAP.
- **✉️ Interactive Contact Mailer**: Prefills email queries with custom input states, ready to transmit signals through the user's system mail client.

## 🏃 Getting Started

To run this project locally, make sure you have [Node.js](https://nodejs.org) installed on your system.

### 1. Clone the repository
```bash
git clone https://github.com/sudip-005/portfolio.git
cd portfolio
```

### 2. Install dependencies
```bash
npm install
```

### 3. Start the development server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### 4. Build for production
```bash
npm run build
```
